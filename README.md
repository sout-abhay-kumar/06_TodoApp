A simple and elegant To-Do List web application built with **React** and **Context API**.  
This app allows you to **add, edit, delete, and mark tasks as complete**, while also saving them in **localStorage** so they persist even after refreshing the page.

---

##  Features
-  **Add Tasks** ‚Äì Enter a task and add it to the list.
-  **Mark as Complete** ‚Äì Toggle tasks between complete and incomplete.
-  **Edit Tasks** ‚Äì Switch tasks to editable mode and update them.
-  **Save Tasks** ‚Äì Tasks are stored in **localStorage**, so they don‚Äôt disappear after refresh.
-  **Delete Tasks** ‚Äì Remove tasks easily.
-  **Modern UI** ‚Äì Styled with Tailwind CSS.

---

## Tech Stack
- **React.js**
- **Context API (for state management)**
- **Tailwind CSS (for styling)**
- **localStorage (for persistence)**

##  Installation

1. Clone this repository:
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>


2. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your browser and go to http://localhost:5713 to see the app.
